To run this project, you will need to add the following environment variables to your .env file

`PORT`
`MONGODB_URI`
`SECRET_KEY`
`NODE_ENV`

## Run Locally

Clone the project

```bash
  git clone https://github.com/fathimathjazeera/ChitChat.git
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
